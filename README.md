# Deep Learning Analysis of the Spectral Dataset

This project aims to perform a Deep Learning analysis of the spectral dataset, including the associated species. 

## Context

- **Required Expertise :**  
  The objective is to leverage expertise in AI and data science to carry out an in-depth analysis of the dataset.  
- **Project Phase :**  
  This analysis takes place in the definition and evaluation of a Deep Learning model.

## Objectives of the Python Script or Python Notebook

The Python script to be generated should:
- Propose a sequential model (or an ensemble of candidate models).
- Define a loss function and a Cross Validation plan analysis (training - validation).
- Optimize the hyperparameters for each model.
- Evaluate and select best model for prediction.
- Produce various plots and criteria to support your work.
  

## Key Points to Address

- **Data Loading and Preparation :**  
  Implementation of the data loading pipeline and preparation for analysis.
- **Model definition :**  
  Try multiple perceptron, convolutional models or inception models or transformers : you can find codes and inspiration at https://huggingface.co/models.
- **Model Training and Validation :**  
  - Plotting of Loss curves (training and validation).
  - Hyperparameters optimisation (on a grid).
-  **Model Selection**
  Select best predicting model (define your criteria, accuracy)
- **Documentation and Explanations :**  
  Inclusion of detailed comments and explanations in the code to facilitate understanding and reuse of the script.
